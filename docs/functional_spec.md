# 세부 기능 정의

## 로그인 기능

### [Task 1] 개인화 프로필 생성 (회원가입)
- **Input:** ID(닉네임), 성별, 연령, 피부 타입, 피부 고민(다중 선택).
- **Logic:** ID중복확인 후 입력된 정보를 customers.json에 신규 레코드로 추가.
- **Output:** "OO님을 위한 맞춤형 에이전트 세팅이 완료되었습니다."

### [Task 2] 기존 프로필 로드 (로그인)
- **Input:** 생성했던 ID 입력.
- **Logic:** customers.json에서 해당 ID를 조회하여 유저의 피부 타입과 과거 로그(이력)를 시스템에 캐싱(Caching).
- **Output:** 이전 활동 내역을 바탕으로 한 인사이트 제시(예: "지난번에 보셨던 진정 앰플은 어떠셨나요?").

### [Task 3] 세션 종료 (로그아웃)
- **Input:** 로그아웃 버튼 클릭.
- **Logic:** 현재 세션의 유저 상태값만 초기화(데이터는 DB에 유지).
- **Output:** 메인 로그인 화면으로 리다이렉트.
